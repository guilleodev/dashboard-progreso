# Dashboard - Progreso
Un pequeño dashboard donde muestro el progreso de mis proyectos personales de forma visual y organizada, así como su descripción, categoría, etc.
El objetivo es tener un panel simple que cargue los proyectos desde un archivo JSON y muestre su estado, categoría y nivel de avance mediante una barra de progreso.

---

# Características (hasta ahora)
- Carga de proyectos desde projects.json
- Tarjetas generadas dinámicamente con JavaScript 
- Barra de progreso por proyecto
- Milestones personalizables (texto y posición)
